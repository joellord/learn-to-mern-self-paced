# Introduction

This is the content for the Learn to MERN workshop. It is a full-stack web development workshop that will teach you how to build a full-stack web application using the MERN stack.

## What do you want to learn?

Take a minute to share with the group what you want to learn from this workshop. This will help us tailor the workshop to your needs.

## This is a safe space

We are all here to learn and grow. Please be respectful of others and their learning process. If you have any questions, please ask them. If you have any concerns, please share them. We are all here to help each other.
